Interactive Timeline

1969: The Birth of ARPANET

ARPANET was the first operational packet switching network and precursor to the modern internet. First packet-switched message between UCLA and SRI.

1973: TCP/IP

Vint Cerf and Bob Kahn propose the TCP/IP protocol suite.

1983: The Internet is Born

ARPANET officially adopts TCP/IP — modern Internet begins. The word “Internet” (interconnected networks) became widely used.

1991: World Wide Web

Tim Berners-Lee launches the first web page.

2007: Mobile Internet

The iPhone revolutionizes mobile access to the Internet.
